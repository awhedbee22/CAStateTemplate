About

The CA State Template Wordpress Theme was created for CA state agencies/departments that are looking to migrate to or update a current Wordpress site. Wordpress is one of the leading CMS in the world. It is a free platform that offers limitless capabilities, security and options. This theme was created off of the CA State Template V4, which offers almost all of the features the State Template V4 offers and then some! Being a open-sourced project, we hope to have many updates that will bring even more features.

Plugins

Wordpress offers a plethora of plugins to help you get up and going without much programming knowledge. Here is a list of a few we find helpful.

  • Contact Form 7 - Simple but flexible contact form.
  • Display Posts Shortcode - Shortcode generator to display specific posts on page.
  • HTML Editor Syntax Highlighter - Adds syntax highlighting in the WordPress post HTML/text editor.
  • Members - A user and role management plugin that puts you in full control of your site's permissions. This plugin allows you to edit your roles and their capabilities, clone existing roles, assign multiple roles per user, block post content, or even make your site completely private.
  • Tab Override - Causes tabs to function as they would in a text editor rather than a browser.
  • WooSidebars - Replace widget areas in your theme for specific pages, archives and other sections of WordPress.
  • Wp Mail Bank - WP Mail Bank reconfigures the wp_mail() function and make it more enhanced.
  • WP Smush - Reduce image file sizes, improve performance and boost your SEO using the free WPMU DEV WordPress Smush API.
